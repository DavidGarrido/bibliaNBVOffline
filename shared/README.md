# Estudios compartidos

El administrador sube aquí los archivos JSON exportados desde la app.

## Formato
Cada archivo debe ser un JSON exportado desde la opción **Exportar estudios** de la app:
```json
{
  "version": 1,
  "exportedAt": "2024-01-01T00:00:00.000Z",
  "studies": [ ... ]
}
```

Un archivo puede contener uno o varios estudios.

## Cómo subir
1. Ir a este repositorio en GitHub
2. Entrar a la carpeta `shared/`
3. Clic en **Add file → Upload files**
4. Subir el archivo `.json` exportado desde la app
